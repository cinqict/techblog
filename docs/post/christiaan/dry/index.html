Commandline Awesome: Dry

Sep 7, 2017 #shell #command line #cool #docker #dry

Een handige tool om te gebruiken terwijl je met Docker aan het werk bent is Dry.

Dry is een grafische terminal app die het gebruik en beheer van Docker duidelijk weergeeft. Dry geeft realtime informatie over jouw draaiende containers en biedt eenvoudige mogelijkheden om bijvoorbeeld images en containers op te ruimen.
